## 1. Vue3 基础语法

### 1.1 组合式 API (Composition API)

```html
<template>
  <div>{{ message }}</div>
  <button @click="handleClick">点击</button>
</template>

<script setup>
  import { ref, reactive, computed, onMounted } from "vue";

  // 响应式数据
  const message = ref("Hello Vue3");
  const state = reactive({
    count: 0,
    name: "张三",
  });

  // 方法
  const handleClick = () => {
    message.value = "Hello World";
    state.count++;
  };

  // 计算属性
  const doubleCount = computed(() => state.count * 2);

  // 生命周期
  onMounted(() => {
    console.log("组件挂载完成");
  });
</script>
```

### 1.2 响应式系统

#### ref 和 reactive

```javascript
import { ref, reactive, toRefs, readonly } from "vue";

// ref: 基础类型响应式
const count = ref(0);
console.log(count.value); // 访问值

// reactive: 对象响应式
const state = reactive({
  name: "张三",
  age: 18,
});

// toRefs: 将 reactive 对象转换为 refs
const { name, age } = toRefs(state);

// readonly: 只读响应式
const original = reactive({ count: 0 });
const copy = readonly(original);
```

#### ref vs reactive

|     对比项     | ref               | reactive          |
| :------------: | :---------------- | :---------------- |
|    支持类型    | 任意类型          | 对象/数组         |
|    访问方式    | `.value`          | 直接访问          |
| 解构丢失响应式 | 会（需用 toRefs） | 会（需用 toRefs） |

### 1.3 常用指令

```html
<template>
  <!-- 文本插值 -->
  <span>{{ message }}</span>

  <!-- v-bind 绑定属性 -->
  <div :id="dynamicId" :class="{ active: isActive }"></div>

  <!-- v-on 绑定事件 -->
  <button @click="handleClick" @mouseenter="onEnter">点击</button>

  <!-- v-if 条件渲染 -->
  <p v-if="show">显示</p>
  <p v-else>隐藏</p>

  <!-- v-show 显示隐藏 -->
  <p v-show="show">展示</p>

  <!-- v-for 列表渲染 -->
  <li v-for="(item, index) in items" :key="item.id">
    {{ index }} - {{ item.name }}
  </li>

  <!-- v-model 双向绑定 -->
  <input v-model="message" />

  <!-- v-slot 插槽 -->
  <MyComponent>
    <template #header>头部</template>
  </MyComponent>

  <!-- 动态绑定 -->
  <div :[attr]="value">动态属性</div>
</template>
```

### 1.4 计算属性与侦听器

```javascript
import { ref, computed, watch, watchEffect } from "vue";

const count = ref(0);
const user = ref({ name: "张三", age: 18 });

// 计算属性
const doubleCount = computed(() => count.value * 2);

// 侦听器
watch(count, (newVal, oldVal) => {
  console.log(`count 变化: ${oldVal} -> ${newVal}`);
});

// 深度监听
watch(
  user,
  (newVal) => {
    console.log("用户变化:", newVal);
  },
  { deep: true }
);

// 立即执行监听
watchEffect(() => {
  console.log("立即执行:", count.value * 2);
});
```

### 1.5 样式绑定

```html

  <!-- 对象语法 -->

  <!-- 数组语法 -->

  <!-- 内联样式 -->

<script setup>

  const hasError = false;
```

## 2. 组件通信

### 2.1 父子组件通信

#### 父 → 子：props

```vue
<!-- Parent.vue -->
<ChildComponent :message="parentMsg" :count="parentCount" />
```

```vue
<!-- Child.vue -->
<script setup>
defineProps({
  message: {
    type: String,
    required: true,
    default: "默认值",
  },
  count: {
    type: Number,
    default: 0,
  },
});
</script>
```

#### 子 → 父：defineEmits

```vue
<!-- Child.vue -->
<template>
  <button @click="sendToParent">发送给父组件</button>
</template>

<script setup>
const emit = defineEmits(["custom-event"]);

const sendToParent = () => {
  emit("custom-event", "数据");
};
</script>
```

```vue
<!-- Parent.vue -->
<ChildComponent @custom-event="handleEvent" />
```

### 2.2 跨级组件通信

#### provide / inject

```vue
<!-- 祖先组件 -->
<script setup>
import { provide, ref } from "vue";

const theme = ref("dark");
provide("theme", theme);
provide("user", { name: "张三" });
</script>
```

```vue
<!-- 后代组件 -->
<script setup>
import { inject } from "vue";

const theme = inject("theme");
const user = inject("user");
</script>
```

### 2.3 兄弟组件通信

#### Event Bus（需自定义）

```javascript
// eventBus.js
import { reactive } from "vue";

export const eventBus = reactive({});

export const emit = (event, data) => {
  if (eventBus[event]) {
    eventBus[event].forEach((callback) => callback(data));
  }
};

export const on = (event, callback) => {
  if (!eventBus[event]) {
    eventBus[event] = [];
  }
  eventBus[event].push(callback);
};

export const off = (event, callback) => {
  if (eventBus[event]) {
    eventBus[event] = eventBus[event].filter((cb) => cb !== callback);
  }
};
```

### 2.5 组件通信方式对比

|       方式        | 适用范围   | 特点                   |
| :---------------: | :--------- | :--------------------- |
| props/defineEmits | 父子组件   | 最常用，类型安全       |
|  provide/inject   | 祖先与后代 | 跨级通信，无需层层传递 |
|     Event Bus     | 任意组件   | 简单，但大型项目难维护 |

## 3. 生命周期

### 3.1 生命周期图解

```
setup → onBeforeMount → onMounted →
onBeforeUpdate → onUpdated →
onBeforeUnmount → onUnmounted
```

### 3.2 组合式 API 生命周期钩子

| 选项式 API      | 组合式 API        | 说明         |
| :-------------- | :---------------- | :----------- |
| `beforeCreate`  | `setup`           | 实例初始化   |
| `created`       | `setup`           | 实例创建完成 |
| `beforeMount`   | `onBeforeMount`   | 挂载前       |
| `mounted`       | `onMounted`       | 挂载完成     |
| `beforeUpdate`  | `onBeforeUpdate`  | 更新前       |
| `updated`       | `onUpdated`       | 更新完成     |
| `beforeDestroy` | `onBeforeUnmount` | 销毁前       |
| `destroyed`     | `onUnmounted`     | 销毁完成     |
| `errorCaptured` | `onErrorCaptured` | 错误捕获     |

### 3.3 生命周期示例

```vue
<script setup>
import {
  onMounted,
  onBeforeMount,
  onBeforeUpdate,
  onUpdated,
  onBeforeUnmount,
  onUnmounted,
} from "vue";

let timer = null;

onBeforeMount(() => {
  console.log("挂载前");
});

onMounted(() => {
  console.log("挂载完成");
  timer = setInterval(() => {
    console.log("定时器运行中");
  }, 1000);
});

onBeforeUpdate(() => {
  console.log("更新前");
});

onUpdated(() => {
  console.log("更新完成");
});

onBeforeUnmount(() => {
  console.log("销毁前");
  clearInterval(timer);
});

onUnmounted(() => {
  console.log("销毁完成");
});
</script>
```

## 4. Vue3 与 Vue2 的区别

### 4.1 核心架构

|   对比项   | Vue2                  | Vue3                   |
| :--------: | :-------------------- | :--------------------- |
| 响应式原理 | Object.defineProperty | Proxy                  |
|  源码组织  | Monorepo              | Monorepo（重构）       |
|    模板    | 只能有一个根节点      | 可以有多个根节点       |
|  类型支持  | TypeScript 支持不完善 | 更好的 TypeScript 支持 |

### 4.2 响应式系统

**Vue2：** 使用 `Object.defineProperty`，需要深度监听

```javascript
// Vue2
data() {
  return {
    user: {
      name: '张三'
    }
  }
}

// 需要深度监听才能检测到属性变化
watch: {
  user: {
    handler() {},
    deep: true
  }
}
```

**Vue3：** 使用 `Proxy`，自动深度响应

```javascript
// Vue3
const user = reactive({
  name: "张三",
});

// 自动深度响应
user.name = "李四"; // 响应式变化
```

### 4.3 组合式 API vs 选项式 API

**Vue2 选项式 API：**

```javascript
export default {
  data() {
    return { count: 0 };
  },
  methods: {
    handleClick() {
      this.count++;
    },
  },
  computed: {
    doubleCount() {
      return this.count * 2;
    },
  },
  watch: {
    count(newVal) {
      console.log(newVal);
    },
  },
};
```

**Vue3 组合式 API：**

```javascript
<script setup>
  import {(ref, computed, watch)} from 'vue' const count = ref(0) const
  doubleCount = computed(() => count.value * 2) const handleClick = () =>{" "}
  {count.value++}
  watch(count, (newVal) => {console.log(newVal)})
</script>
```

> **组合式 API 优势：**
>
> 1. 更好的逻辑复用
> 2. 更好的类型推导
> 3. 更好的代码组织（相关逻辑集中）
> 4. 更好的 tree-shaking 支持

### 4.4 生命周期

| Vue2            | Vue3              |
| :-------------- | :---------------- |
| `beforeCreate`  | `setup`           |
| `created`       | `setup`           |
| `beforeMount`   | `onBeforeMount`   |
| `mounted`       | `onMounted`       |
| `beforeUpdate`  | `onBeforeUpdate`  |
| `updated`       | `onUpdated`       |
| `beforeDestroy` | `onBeforeUnmount` |
| `destroyed`     | `onUnmounted`     |

### 4.5 组件通信

|  对比项  | Vue2                     | Vue3           |
| :------: | :----------------------- | :------------- |
| 父 → 子  | props                    | defineProps    |
| 子 → 父  | $emit                    | defineEmits    |
|   跨级   | provide/inject (Vue2.2+) | provide/inject |
| 状态管理 | Vuex                     | Pinia          |
| 生命周期 | 选项式                   | 组合式 API     |

### 4.6 新增特性

**Vue3 新增特性：**

1. **Teleport：** 传送门，将组件渲染到任意位置

```vue
<Teleport to="body">
  <Modal />
</Teleport>
```

2. **Suspense：** 异步组件加载

```vue
<Suspense>
  <template #default>
    <AsyncComponent />
  </template>
  <template #fallback>
    <Loading />
  </template>
</Suspense>
```

3. **Fragment：** 片段，支持多根节点

```vue
<template>
  <header>头部</header>
  <main>内容</main>
  <footer>底部</footer>
</template>
```

4. **v-memo：** 缓存模板

```vue
<div v-memo="[count]">
  {{ count }}
</div>
```

5. **CSS 变量：** 支持在 Vue 中使用 CSS 变量

```vue
<template>
  <div class="card">卡片</div>
</template>

<style scoped>
.card {
  --card-bg: v-bind("theme.bg");
}
</style>
```

### 4.7 性能优化

|  对比项  | Vue2      | Vue3                   |
| :------: | :-------- | :--------------------- |
| 虚拟 DOM | VNode 1.0 | VNode 2.0（静态标记）  |
| 编译优化 | 无        | Block Tree + PatchFlag |
| 打包体积 | 较大      | 较小（tree-shaking）   |
| 渲染性能 | 一般      | 提升约 50%             |

### 4.8 迁移建议

1. **新项目：** 推荐使用 Vue3 + Composition API + Pinia
2. **老项目：** 可渐进式迁移，使用 `vue-composition-api`
3. **TypeScript：** Vue3 原生支持，Vue2 需要额外配置
