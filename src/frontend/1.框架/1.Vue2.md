## 1. Vue2 基础语法

### 1.1 模板语法

```html
<div id="app">
  <!-- 文本插值 -->
  <span>Message: {{ msg }}</span>

  <!-- 原始 HTML -->
  <div v-html="rawHtml"></div>

  <!-- 属性绑定 -->
  <div v-bind:id="dynamicId"></div>
  <div :class="className"></div>

  <!-- 事件绑定 -->
  <button v-on:click="handleClick">点击</button>
  <button @click="handleClick">点击</button>

  <!-- 条件渲染 -->
  <p v-if="show">显示</p>
  <p v-show="show">展示</p>

  <!-- 列表渲染 -->
  <li v-for="item in items" :key="item.id">{{ item.name }}</li>

  <!-- 表单绑定 -->
  <input v-model="message" />
</div>
```

```javascript
new Vue({
  el: "#app",
  data: {
    msg: "Hello Vue",
    rawHtml: '<span style="color:red">红色文本</span>',
    dynamicId: "box",
    className: "active",
    show: true,
    items: [
      { id: 1, name: "苹果" },
      { id: 2, name: "香蕉" },
    ],
    message: "",
  },
  methods: {
    handleClick() {
      console.log("点击了");
    },
  },
});
```

### 1.2 常用指令

|              指令               | 说明                        |
| :-----------------------------: | :-------------------------- |
|            `v-bind`             | 绑定属性，简写为 `:`        |
|             `v-on`              | 绑定事件，简写为 `@`        |
| `v-if` / `v-else-if` / `v-else` | 条件渲染（真正的 DOM 操作） |
|            `v-show`             | 显示隐藏（display 控制）    |
|             `v-for`             | 列表渲染，必须加 key        |
|            `v-model`            | 双向数据绑定                |
|            `v-slot`             | 插槽，简写为 `#`            |
|            `v-once`             | 只渲染一次                  |
|             `v-pre`             | 跳过编译                    |

### 1.3 计算属性与侦听器

```javascript
new Vue({
  el: "#app",
  data: {
    firstName: "张",
    lastName: "三",
    user: { name: "张三", age: 18 },
  },
  computed: {
    // 计算属性，基于依赖缓存
    fullName() {
      return this.firstName + this.lastName;
    },
  },
  watch: {
    // 侦听器，监听数据变化
    firstName(newVal, oldVal) {
      console.log(`姓发生变化: ${oldVal} -> ${newVal}`);
    },
    // 深度监听对象
    user: {
      handler(newVal) {
        console.log("用户信息变化:", newVal);
      },
      deep: true,
    },
  },
});
```

> **计算属性 vs 侦听器**
>
> - 计算属性：用于计算衍生数据，基于依赖缓存
> - 侦听器：用于执行异步操作或开销较大的操作

### 1.4 过滤器

```html
<!-- 在双花括号中 -->
{{ message | capitalize }}

<!-- 在 v-bind 中 -->
<div :id="id | formatId"></div>
```

```javascript
new Vue({
  filters: {
    capitalize(value) {
      if (!value) return "";
      return value.charAt(0).toUpperCase() + value.slice(1);
    },
  },
});
```

## 2. 组件通信

### 2.1 父子组件通信

#### 父 → 子：props

```javascript
// 父组件
<ChildComponent :message="parentMsg" :count="parentCount" />

// 子组件
props: {
  message: {
    type: String,
    required: true,
    default: '默认值'
  },
  count: {
    type: Number,
    default: 0
  }
}
```

#### 子 → 父：$emit

```html
<!-- 子组件 -->
<button @click="sendToParent">发送给父组件</button>
```

```javascript
// 子组件
methods: {
  sendToParent() {
    this.$emit('custom-event', '数据')
  }
}
```

```html
<!-- 父组件 -->
<ChildComponent @custom-event="handleEvent" />
```

```javascript
// 父组件
methods: {
  handleEvent(data) {
    console.log('收到:', data)
  }
}
```

### 2.2 跨级组件通信

#### provide / inject

```javascript
// 祖先组件
provide() {
  return {
    theme: this.theme,
    user: this.user
  }
}

// 后代组件
inject: ['theme', 'user']
```

#### $parent / $children

```javascript
// 子组件获取父组件
this.$parent.title;

// 父组件获取子组件
this.$children[0].someMethod();
```

### 2.3 兄弟组件通信（Event Bus）

```javascript
// 创建一个事件总线
export const EventBus = new Vue();

// 组件 A 发送事件
EventBus.$emit("share-data", data);

// 组件 B 监听事件
EventBus.$on("share-data", (data) => {
  console.log("收到:", data);
});

// 组件 B 销毁时移除监听
EventBus.$off("share-data");
```

### 2.5 组件通信方式对比

|       方式        | 适用范围   | 特点                     |
| :---------------: | :--------- | :----------------------- |
|    props/$emit    | 父子组件   | 最常用，简单直接         |
| $parent/$children | 父子组件   | 耦合度高，不推荐         |
|  provide/inject   | 祖先与后代 | 跨级通信，无需层层 props |
|     Event Bus     | 任意组件   | 简单，但大型项目难维护   |
|       Vuex        | 任意组件   | 功能强大，适合大型应用   |

## 3. 生命周期

### 3.1 生命周期图解

```
beforeCreate → created → beforeMount → mounted →
beforeUpdate → updated → beforeDestroy → destroyed
```

### 3.2 各个钩子详解

|      钩子       | 阶段         | 说明                             |
| :-------------: | :----------- | :------------------------------- |
| `beforeCreate`  | 实例初始化   | data、methods 不可用             |
|    `created`    | 实例创建完成 | 可访问 data、methods，$el 不可用 |
|  `beforeMount`  | 挂载前       | 模板编译完成，尚未渲染           |
|    `mounted`    | 挂载完成     | DOM 已渲染，可获取 $el           |
| `beforeUpdate`  | 更新前       | 数据变化，DOM 尚未更新           |
|    `updated`    | 更新完成     | DOM 已更新，避免再次修改数据     |
| `beforeDestroy` | 销毁前       | 实例仍可用，清理定时器等         |
|   `destroyed`   | 销毁完成     | 解除绑定，销毁子组件             |

### 3.3 生命周期示例

```javascript
new Vue({
  data: {
    message: "Hello",
  },
  beforeCreate() {
    console.log("beforeCreate:", this.message); // undefined
  },
  created() {
    console.log("created:", this.message); // Hello
    this.timer = setInterval(() => {
      console.log("定时器运行中");
    }, 1000);
  },
  mounted() {
    console.log("mounted:", this.$el); // DOM 元素
    this.fetchData();
  },
  beforeDestroy() {
    console.log("beforeDestroy: 清理资源");
    clearInterval(this.timer);
  },
});
```
