## 前言

前提环境：

- 安装 node 和 npm ，node 版本在 22 以上

以下为 mac mini 安装 brew ，并通过 brew 安装 node

安装 brew：

```shell
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

安装 node：

```shell
brew install node@22
```

将 node@22 链接到系统

```shell
brew link node@22 --force --overwrite
```

查看版本

```shell
node --version
npm --version
```

## 安装 OpenClaw

```shell
npm install -g openclaw@latest
```

初始化

```shell
openclaw onboard
```

## 接入微信

```shell
npx -y @tencent-weixin/openclaw-weixin-cli@latest install
```

可能会报证书错误，运行以下命令，禁用 TLS/SSL 证书验证，再执行上面代码

```shell
NODE_TLS_REJECT_UNAUTHORIZED=0
```

安装插件完毕后会弹出一个二维码，打开微信并绑定

重启网关后，到后台的**频道**上看是否对接微信的频道是否在运行，运行中即接入成功

```shell
openclaw gateway restart
```

## 接入飞书

新版本的 OpenClaw 已内置飞书插件，只需启用该插件

```shell
openclaw plugins enable feishu
```

在飞书开放平台创建机器人，添加**消息接收**权限，发布版本应用

添加飞书频道

```shell
openclaw channels add
```

选择 "Feishu/Lark (飞书)"，后弹出一个二维码，打开飞书 App ，扫描并选择刚才添加的机器人

重启网关后，到后台的**频道**上看是否对接微信的频道是否在运行，运行中即接入成功

```shell
openclaw gateway restart
```
