## 前言

**Claude Code** 是 Anthropic 推出的一款命令行编码辅助工具。

简单来说，它不只是一个能在终端里和你聊天的机器人，它更像是一个拥有“操作权限”的 AI 结对编程伙伴。它能直接读取我们的本地代码库、编写文件、运行终端命令、执行测试，甚至直接帮我们提交 Git Commit 和 Pull Request。

## 安装

前提环境：Node 、Git

设置国内镜像源（加速下载）

```shell
# 设置国内镜像源（加速下载）
npm config set registry https://registry.npmmirror.com

# 安装稳定版本（2.1.112，npmmirror 可用的最新版）
npm install -g @anthropic-ai/claude-code@2.1.112

# 切回官方源（重要：避免 Claude Code 自动更新时拉到不完整的平台包）
npm config set registry https://registry.npmjs.org
```

### win

```shell
# 设置系统环境变量（需要管理员权限）
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://api.minimaxi.com/anthropic", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_AUTH_TOKEN", "密钥", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_MODEL", "MiniMax-M2.7", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_SMALL_FAST_MODEL", "MiniMax-M2.7", "User")
# 重启PowerShell使配置生效
Exit
```

### mac

将以下内容添加添加到 `~/.bashrc` 或 `~/.zshrc`

```shell
# 1. 设置 API 地址
export ANTHROPIC_BASE_URL="https://api.minimaxi.com/anthropic"

# 2. 设置中转 API Key
export ANTHROPIC_AUTH_TOKEN="密钥"

# 4. 指定模型
export ANTHROPIC_MODEL="MiniMax-M2.7"
export ANTHROPIC_SMALL_FAST_MODEL="MiniMax-M2.7"
```

## 使用

常用的交互命令如下：

- /：操作型命令

| **命令** | 作用                                                                      |
| -------- | ------------------------------------------------------------------------- |
| /init    | 自动为项目生成 `CLAUDE.md`，用于存储编码规范和项目上下文                  |
| /compact | 压缩上下文                                                                |
| /clear   | 清空对话                                                                  |
| /btw     | 允许用户在 Claude Code 执行主任务时，提出一个旁路问题而不干扰当前工作流程 |
| /plan    | 进入规划模式                                                              |
| /model   | 切换模型                                                                  |
| /context | 查看上下文使用情况                                                        |
