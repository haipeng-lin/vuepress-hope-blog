## 前言

`n8n`是一个强大的**工作流自动化工具**，它采用了基于节点的“低代码/无代码”可视化界面，让你能够将不同的应用程序、数据库和 API 串联起来，从而实现各种业务流程的自动化

## n8n 安装

### docker 方式

拉取最新镜像

```shell
docker pull n8nio/n8n:latest
```

查看版本：2.1.2

```shell
docker exec -it n8n n8n --version
```

基于镜像运行容器

```shell
docker run -d \
  --name n8n \
  --restart always \
  -p 5678:5678 \
  -v /home/n8n:/home/node/.n8n \
  -e WEBHOOK_URL=http://8.155.33.36:5678/ \
  n8nio/n8n
```

查看 n8n 容器日志：

```shell
docker logs n8n
```

测试 ping 是否成功：

```shell
curl -I http://localhost:5678
```

## 汉化 8n8

**GitHub 项目地址:** [https://github.com/other-blowsnow/n8n-i18n-chinese](https://github.com/other-blowsnow/n8n-i18n-chinese)

1. 下载与你的 n8n 版本相对应的 `editor-ui.tar.gz` 文件
2. 解压缩并将 editor-ui 文件夹传输到服务器上 /home/han-hua/n8n/editor-ui
3. 停止并删除原来的 n8n 容器，重新启动容器

```shell
docker run -d \
--name n8n \
-p 5678:5678 \
-v /home/han-hua/n8n/editor-ui/dist:/usr/local/lib/node_modules/n8n/node_modules/n8n-editor-ui/dist \
-v /home/n8n:/home/node/.n8n \
-e N8N_DEFAULT_LOCALE=zh-CN \
-e N8N_SECURE_COOKIE=false \
-e WEBHOOK_URL=http://8.155.33.36:5678/ \
n8nio/n8n
```

4. 查看日志

   ```shell
   docker logs n8n
   ```

5. 访问 IP 地址+5678，成功访问
