## 定义

`LangGraph4j `是 一个用于构建有状态、多步骤 AI 工作流的`LangGraph `框架 的`Java `版本

应用场景

- 智能客服：意图识别 → 知识库查询 → 生成回答 → 满意度检查 → 循环或结束
- 旅行助手：查天气 → 推荐景点 → 推荐美食 → 生成小红书文案
- 代码分析：解析代码 → 安全扫描 → 性能评估 → 生成报告
- 多工具调用：先搜索 → 再计算 → 再查询数据库 → 汇总回答
- Agent 循环：LLM 生成 Action → 执行工具 → 结果反馈 → 再次决策

## 核心概念

- `StateGraph`：状态图构建器，定义节点、边、条件分支组成有向图
- `AgentState`：状态基类，通过 .value(key) 获取状态值，.put(key, value) 更新状态
- `Node`：节点，每个节点执行一个 LLM 调用或工具
- `Edge`：边，节点间的连接，控制执行流程
- `START / END`：图的生命周期起止点
- `CompiledGraph`：编译后的可执行图，调用 .invoke() 触发执行
- `ConditionalEdges`：条件边，根据状态动态选择下一个节点

## 基础工作流

流程如下：

### 定义状态类

定义状态类，继承`AgentState`类

```java
public class MyState extends AgentState {
    // 状态数据的 Key ，在状态图中存储和传递数据
    public static final String INPUT = "input";
    public static final String OUTPUT = "output";

    public MyState(Map<String, Object> initData) { super(initData); }

    public String getInput() { return (String) value(INPUT).orElse(""); }
    public String getOutput() { return (String) value(OUTPUT).orElse(""); }
}
```

### 创建节点服务类

```java
public class MyNodeService {
    // 定义节点A
    public Map<String, Object> nodeA(MyState state) {
        // 业务逻辑，返回要更新到状态的KV
        return Map.of("key", "value");
    }
}
```

### 构建编译图

作用：

- 注入节点服务实例
- 构建图，添加节点、边
- 编译图
- 提供 `invoke`方法供外部执行已编译完的工作图

```java
public class MyGraphExecutor {
    private final MyNodeService nodeService;
    private final CompiledGraph<MyState> graph;

    public MyGraphExecutor(MyNodeService nodeService) {
        this.nodeService = nodeService;
        this.graph = new GraphBuilder(nodeService).build().compile();
    }

    class GraphBuilder {
        StateGraph<MyState> build() {
            return new StateGraph<>(MyState::new)
                .addNode("nodeA", node_async(nodeService::nodeA))  // 添加节点
                .addEdge(START, "nodeA")                           // 定义边
                .addEdge("nodeA", END);                             // 连接到结束
        }
    }

    // 4. 执行图
    public MyState invoke(String input) {
        return graph.invoke(Map.of(MyState.INPUT, input)).orElse(...);
    }
}
```

```java
public class SimpleState extends AgentState {

    }

    }

    public String getWeather() {
        return (String) value(WEATHER).orElse("");

    public String getResponse() {
        return (String) value(RESPONSE).orElse("");
```

**2、创建节点服务类**

```java
@Service
public class SimpleGraphNodeService {

    private final ChatClient chatClient;

    public SimpleGraphNodeService(OpenAiChatModel chatModel) {
        this.chatClient = ChatClient.builder(chatModel).build();
    }

    // 天气节点
    public Map<String, Object> weatherNode(SimpleState state) {
        String prompt = """
                根据用户输入的地点，生成一句关于该地点天气的描述。
                地点：{input}
                """;
        PromptTemplate template = new PromptTemplate(prompt);
        String result = chatClient.prompt()
                .user(template.render(Map.of("input", state.getInput())))
                .call()
                .content();
        return Map.of(SimpleState.WEATHER, result);
    }

    // 响应节点
    public Map<String, Object> responseNode(SimpleState state) {
        String prompt = """
                根据天气信息，用温暖的语气给用户一个简短的回复和建议。
                天气：{weather}
                """;
        PromptTemplate template = new PromptTemplate(prompt);
        String result = chatClient.prompt()
                .user(template.render(Map.of("weather", state.getWeather())))
                .call()
                .content();
        return Map.of(SimpleState.RESPONSE, result);
    }
}
```

**构建编译图**

```java
@Service
public class SimpleGraphExecutor {
    private static final Logger log = LoggerFactory.getLogger(SimpleGraphExecutor.class);

    public static final String WEATHER_NODE = "weatherNode";
    public static final String RESPONSE_NODE = "responseNode";

    private final SimpleGraphNodeService nodeService;
    private final CompiledGraph<SimpleState> compiledGraph;

    public SimpleGraphExecutor(SimpleGraphNodeService nodeService) throws GraphStateException {
        this.nodeService = nodeService;
        this.compiledGraph = new GraphBuilder().build().compile();
        printGraph();
    }

    public SimpleState invoke(String input) {
        return compiledGraph.invoke(Map.of(SimpleState.INPUT, input))
                .orElseGet(() -> new SimpleState(Map.of("error", "No response")));
    }

    class GraphBuilder {
        public StateGraph<SimpleState> build() throws GraphStateException {
            return new StateGraph<>(SimpleState::new)
                    .addNode(WEATHER_NODE, node_async(nodeService::weatherNode))
                    .addNode(RESPONSE_NODE, node_async(nodeService::responseNode))
                    .addEdge(START, WEATHER_NODE)
                    .addEdge(WEATHER_NODE, RESPONSE_NODE)
                    .addEdge(RESPONSE_NODE, END);
        }
    }

    private void printGraph() {
        try {
            GraphRepresentation rep = compiledGraph.getGraph(GraphRepresentation.Type.MERMAID, "SimpleGraph", true);
            log.info("Mermaid Graph:\n{}", rep.content());
        } catch (Exception e) {
            log.warn("Failed to print graph: {}", e.getMessage());
        }
    }
}
```

## 并行工作流

### 实现

- 实现方式：通过 Fan-Out（分叉） 实现并行：多个节点共享同一个前驱节点时，它们会同时执行
- 举例：

```java
class GraphBuilder {
    public StateGraph<MyState> build() throws GraphStateException {
        return new StateGraph<>(MyState::new)
            // 三个节点并行执行
            .addNode("nodeB", node_async(service::nodeB))
            .addNode("nodeC", node_async(service::nodeC))
            .addNode("nodeD", node_async(service::nodeD))

            // START → B, C, D 同时执行
            .addEdge(START, "nodeB")
            .addEdge(START, "nodeC")
            .addEdge(START, "nodeD")

            // 汇聚到 END
            .addEdge("nodeD", END);
    }
}
```

```java
@Service
public class ParallelGraphExecutor {
    private static final Logger log = LoggerFactory.getLogger(ParallelGraphExecutor.class);

    public static final String TOURISM_NODE = "tourismNode";
    public static final String FOOD_NODE = "foodNode";
    public static final String WEATHER_NODE = "weatherNode";

    private final ParallelNodeService nodeService;
    private final CompiledGraph<ParallelState> compiledGraph;

    public ParallelGraphExecutor(ParallelNodeService nodeService) throws GraphStateException {
        this.nodeService = nodeService;
        this.compiledGraph = new GraphBuilder().build().compile();
        printGraph();
    }

    public ParallelState invoke(String location) {
        return compiledGraph.invoke(Map.of(ParallelState.LOCATION, location))
            .orElseGet(() -> new ParallelState(Map.of("error", "No response")));
    }

    class GraphBuilder {
        public StateGraph<ParallelState> build() throws GraphStateException {
            return new StateGraph<>(ParallelState::new)
                // 三个并行节点
                .addNode(TOURISM_NODE, node_async(nodeService::tourismNode))
                .addNode(FOOD_NODE, node_async(nodeService::foodNode))
                .addNode(WEATHER_NODE, node_async(nodeService::weatherNode))
                // Fan-Out: START 同时触发三个节点
                .addEdge(START, TOURISM_NODE)
                .addEdge(START, FOOD_NODE)
                .addEdge(START, WEATHER_NODE)
                // 汇聚到 END
                .addEdge(TOURISM_NODE, END)
                .addEdge(FOOD_NODE, END)
                .addEdge(WEATHER_NODE, END);
        }
    }

    private void printGraph() {
        try {
            GraphRepresentation rep = compiledGraph.getGraph(GraphRepresentation.Type.MERMAID, "ParallelGraph", true);
            log.info("Mermaid Graph:\n{}", rep.content());
        } catch (Exception e) {
            log.warn("Failed to print graph: {}", e.getMessage());
        }
    }
}
```
